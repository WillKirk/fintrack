# FinTrack

A personal finance dashboard for tracking income, expenses, budgets and savings goals.

Live at [fintrack-wk.vercel.app](https://fintrack-wk.vercel.app) — click "Continue as guest" to try it without signing up.

## Features

- Dashboard with monthly summary, spending breakdown and recent transactions
- Transaction management with CSV import from any bank
- Budget limits per category with progress tracking
- Savings goals with projected completion dates
- Compound interest and savings projection calculator
- Multi-user auth with isolated guest demo accounts

## Tech stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- MySQL (Railway)
- NextAuth.js
- Recharts
- Framer Motion
